# The Three Numbers

## Question

Given three integers. Find out the second biggest integer.

## Input

Given three integers which is A, B and C. Each of numbers are separated with a whitespace. $ (1 \le A, B, C \le 100) $

## Output

Print the second biggest integer.

## Result

### Expected input 1

```
20 30 10
```

### Expected output 1

```
20
```

### Expected input 2

```
30 30 10
```

### Expected output 2

```
30
```

### Expected input 3

```
40 40 40
```

### Expected output 3

```
40
```

### Expected input 4

```
20 10 10
```

### Expected output 4

```
10
```
